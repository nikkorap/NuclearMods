# WSO Yappinator

Get a variety of characters to yap at you while you fly!

* [Download mod](https://github.com/nikkorap/NuclearMods/raw/refs/heads/main/clientside,%20QOL,%20accessibility/WSOYappinator/com.nikkorap.WSOYappinator_2.1.1.dll)
* [Browse Voicepacks](Audio) (to download, click on the desired pack, then click "download raw file" on the right)

Folder structure should look like this:
```
bepinex/plugins/
  └─ WSOYappinator/
    ├─ WSOYappinator_version.dll
    └─ audio/
       ├─ Prez/
       │  ├─ eventPriorities.txt
       │  └─ audiofiles.mp3
       └─ Galaxy/
          ├─ eventPriorities.txt
          └─ audiofiles.wav
```
